# Sanjay S — Portfolio


This repository contains a static multi-page portfolio built with HTML, CSS and vanilla JavaScript. It is optimized for GitHub Pages deployment.


## Structure
- `index.html` — Home
- `about.html` — About & Future Goals
- `projects.html` — Projects
- `education.html` — Education & Certifications
- `contact.html` — Contact (uses Formspree example)
- `assets/css/style.css` — Styles
- `assets/js/main.js` — Interactivity
- `assets/images/profile.jpg` — Add your photo here


## Quick local preview
1. Open the folder in VS Code
2. Install Live Server extension
3. Right-click `index.html` → Open with Live Server


## Deploy to GitHub Pages
1. Push the repository to `sanaimax1321.github.io` remote
2. Enable GitHub Pages in repository Settings → Pages → Branch: main → Root
